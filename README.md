## Installation
1. Install composer dependencies
```
    $ php composer.phar install or composer install
```

2. Set the application Key
```
    $ php artisan key:generate
```

3. Configure .env
```
    DB_CONNECTION=mysql
    DB_HOST=127.0.0.1
    DB_PORT=3306
    DB_DATABASE=database
    DB_USERNAME=user
    DB_PASSWORD=password
```

4. Run migrations
```
    $ php artisan migrate
```
